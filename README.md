## 项目概述


## 技术栈

- 后端：Python + Django
- 前端：Vue.js
- 数据库：MySQL

## 系统角色

- 管理员：负责系统维护和用户管理。
- 用户：享受系统提供的个性化阅读推荐服务。

## 是否前后分离

本项目采用前后端分离的架构。如果原电脑的Node.js版本高于14，将提供远程配置cnpm环境的服务。

## 运行环境

- 后端：Python环境（推荐使用虚拟环境）
- 前端：Node.js环境（版本要求高于14）
- 数据库：MySQL数据库

## 部署步骤

1. 安装后端Python环境和所需库。
2. 配置MySQL数据库。
3. 部署前端Vue项目。
4. 配置前后端通信。
